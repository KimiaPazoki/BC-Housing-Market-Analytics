# SQL Scripts

This folder contains SQL scripts used for data cleaning, preparation, and validation before building the Power BI dashboard.

## Files

### housing-data-cleaning.sql
Checks missing values, duplicate region-year records, and incomplete housing price fields in the original BC housing dataset.

### housing-data-validation.sql
Validates transformed housing tables, housing type values, vacancy rate ranges, region matching, and date fields.

## Purpose
These SQL scripts demonstrate data quality checks and preparation steps used before Power BI modeling and DAX-based KPI reporting.
